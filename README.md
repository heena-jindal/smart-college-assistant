# ⚙️ Installation

## 1️⃣ Clone Repository

```bash
git clone https://github.com/heena-jindal/smart-college-assistant.git

cd smart-college-assistant
```

---

## 2️⃣ Backend Setup

```bash
cd smart-college-backend

python -m venv venv

venv\Scripts\activate

pip install -r requirements.txt
```

Run backend:

```bash
python main.py
```

Backend runs on:

```text
http://localhost:5000
```

---

## 3️⃣ Frontend Setup

```bash
cd smart-college-frontend

npm install
```

Run frontend:

```bash
npm run dev
```

Frontend runs on:

```text
http://localhost:3000
```
